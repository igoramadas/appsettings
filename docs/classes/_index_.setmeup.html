loadFromEnv

  • loadFromEnv(prefix?: string, options?: LoadEnvOptions): any
    • Defined in index.ts:216

    Load settings from environment variables, restricting to the passed prefix. Enviroment settings as variables will be split by underscore to define its tree.

    Parameters

    • Optional prefix: string

      The prefix use to match relevant environment variables. Default is "SMU_", should always end with "_" (underscore).

    • Optional options: LoadEnvOptions

      Load options defining if properties should be overwritten and forced to lowercase.

    Returns any
